# RE-PSCoverDL


### Features

- Portable
- GUI
- Settings file
- Multithreading

### How to use

- Select the desired emulator.
- Choose the output folder for covers (usually pcsx2/covers).
- Select the "gamelist.cache" file (pcsx2/cache).
- Choose your desired cover type.
- Press the start download button.
